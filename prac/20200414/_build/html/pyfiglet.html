smushAmount(buffer=[], curChar=[])¶

Calculate the amount of smushing we can do between this char and the last If this is the first char it will throw a series of exceptions which are caught and cause appropriate values to be set for later.

This differs from C figlet which will just get bogus values from memory and then discard them after.

smushChars(left='', right='')¶

Given 2 characters which represent the edges rendered figlet fonts where they would touch, see if they can be smushed together. Returns None if this cannot or should not be done.
